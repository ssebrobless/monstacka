# MonStacka

Standalone downloadable builds for `MonStacka!`.

## Downloads

Use the latest GitHub Release on this repo for the Windows desktop build:
- `MonStacka!_0.1.0_x64-setup.exe` - recommended Windows installer
- `MonStacka!_0.1.0_x64_en-US.msi` - MSI installer
- `monstacka.exe` - portable executable
- `SHA256SUMS.txt` - checksums for verification

## Notes

- This repo is intended as the standalone build/download home for `MonStacka!`.
- The broader source project currently lives at:
